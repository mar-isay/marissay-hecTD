# HEC TD: Üniversite Mobil Uygulama (Eğitim Asistanı)

## Proje Vizyonu
[cite_start]Bu uygulama, Bilgisayar Mühendisliği 2. sınıf düzeyinde, modern teknolojiler kullanılarak tüm öğrenciler için hem Android hem de iOS platformlarında çalışacak kapsamlı bir eğitim asistanı olarak tasarlanmıştır[cite: 3].

## Teknoloji Yığını (Tech Stack)
* [cite_start]**Mobil:** React Native (Expo) - MacBook Air ve React tecrübesiyle en hızlı çıktı için[cite: 5].
* [cite_start]**Backend:** Node.js (Express.js) - JavaScript ekosisteminde kalarak öğrenme kolaylığı sağlar[cite: 5].
* [cite_start]**Veritabanı:** PostgreSQL (Supabase) - Öğrenci ve ders ilişkileri için yapısal veritabanı[cite: 5].
* [cite_start]**Güvenlik:** Firebase Auth / JWT - Güvenli kullanıcı yönetimi[cite: 5, 6].
