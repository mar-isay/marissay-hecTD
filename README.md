# HEC TD: Üniversite Öğrenci Asistanı
    Mar_isay

## 🚀 Proje Vizyonu
Bu proje, üniversite öğrencilerinin akademik hayatlarını kolaylaştırmayı hedefleyen, modern teknolojilerle geliştirilmiş bir mobil/web uygulamasıdır.

## Temel Özellikler
* [cite_start]**Ders Takibi:** Dönemlik ders programı ve ders içerikleri[cite: 19].
* **Not Sistemi & Transcript:** Sınav sonuçları, harf notları ve genel ortalama (GPA) takibi.
* [cite_start]**Akademik İçerikler:** Ders notları ve materyallere hızlı erişim.
* [cite_start]**QR Kimlik:** Öğrenci kimlik bilgilerine hızlı erişim sağlayan dinamik QR kod modülü[cite: 17].

## Teknoloji Yığını (Tech Stack)
* [cite_start]**Mobil:** React Native & Expo (MacBook Air ve React tecrübesiyle hızlı çıktı almak için)[cite: 5].
* [cite_start]**Backend:** Node.js & Express.js (JavaScript ekosistemiyle uyumlu)[cite: 5].
* [cite_start]**Veritabanı:** PostgreSQL & Supabase (İlişkisel veriler için en doğru tercih)[cite: 5].
* [cite_start]**Kimlik Doğrulama:** Firebase Auth / JWT (Güvenli giriş yönetimi)[cite: 5, 6].
